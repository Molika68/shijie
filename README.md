# 识界 Shijie

多模态 AI 助手：拍照识物、文字生图、历史记录。  
技术栈：Nest.js + Prisma + Taro/React。

## 目录

| 目录 | 说明 |
|------|------|
| `shijie-backend/` | 后端 API |
| `shijie-frontend/` | Taro H5 / 小程序前端 |
| `scripts/` | 本地一键启动 |

## 本地开发

```bash
chmod +x scripts/dev.sh
./scripts/dev.sh
```

- 前端：http://localhost:10086  
- 后端：http://localhost:3000/api  

详细说明见 [README-shijie.md](./README-shijie.md)、[SETUP-NOW.md](./SETUP-NOW.md)。
